## Course Project — Multimodal Tool Retrieval

In this project, we examine an agentic multimodal retrieval system that separates structured data such as API schemas and code from natural language queries. In addition, we will discuss hierarchical contrastive alignment, retrieval based on code, and hard-negative mining methods in the context of ToolBench.
